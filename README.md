# אתר בינה מלאכותית ללימוד

## סקירה כללית
פרויקט ליצירת אתר חינוכי בנושא בינה מלאכותית.

## התקנה והגדרה

### דרישות מערכת
- Node.js (גרסה 18 ומעלה)
- npm או yarn

### הגדרת משתני סביבה
1. העתק את הקובץ `.env.example` לקובץ `.env`
2. ערוך משתני סביבה לפי הצורך:
   ```
   PORT=3000
   NODE_ENV=development
   ```

**הערה**: Claude Code משתמש במפתח קונסול פנימי ולא צריך ANTHROPIC_API_KEY

### הפעלה מקומית
```bash
npm install
npm start
```

## מבנה הפרויקט
- `TASKS.MD` - רשימת משימות והתקדמות
- `CLAUDE.MD` - הנחיות לעבודה
- `.env` - משתני סביבה (לא כלול ב-git)

## מידע נוסף
עבוד לפי רשימת המשימות ב-`TASKS.MD`
